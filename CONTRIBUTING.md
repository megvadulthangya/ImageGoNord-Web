# Contributing to ImageGoNord (Self-Hosted)

Thanks for your interest in improving this project! Since this is a specialized fork for self-hosting, we keep things simple.

## How to Contribute

1.  **Fork the repository.**
2.  **Make your changes.** (Fix bugs, optimize code, remove bloat).
3.  **Submit a Pull Request.**

## Guidelines

* **Keep it Clean:** No tracking, no ads, no external dependencies unless absolutely necessary.
* **Keep it Stable:** The goal is to run this on modest hardware (workers=1).
* **No Politics:** We are here for the code, not for social activism.

## Reporting Bugs
